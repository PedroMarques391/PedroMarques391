## Sobre mim

Sou **Desenvolvedor Full Stack**, com atuação em aplicações **web e mobile** e forte interesse em **arquitetura de software**. Gosto de entender o sistema como um todo, desde a interface até o backend, incluindo modelagem de dados, contratos de API e organização do código.

Já desenvolvi projetos full stack do zero, definindo arquitetura, regras de negócio, integrações e interfaces, sempre priorizando código limpo, escalabilidade, manutenção e boas práticas.

Tenho facilidade em aprender novas tecnologias, gosto de discutir decisões técnicas e acredito que uma boa arquitetura reduz complexidade e aumenta a longevidade do produto.
